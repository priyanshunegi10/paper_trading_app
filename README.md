# 🚀 Crypto Paper Trading App

Ek premium aur full-stack Flutter application jahan users bina asli paisa khoye real-time market data ke sath crypto trading seekh sakte hain. Is app mein users ko virtual $100,000 milte hain trading shuru karne ke liye.

---

## ✨ Key Features

- **🔐 Firebase Authentication**: Secure Sign-up aur Login system (Email/Password).
- **📈 Real-time Market Data**: CoinGecko API ka use karke top 100+ cryptocurrencies ka live price aur data.
- **💰 Virtual Wallet**: Naye users ko $100,000 ka starting balance milta hai.
- **📂 Portfolio Management**: Dekho tumne kaunsa coin kitni quantity mein kharida hai aur tumhara total profit/loss kitna hai.
- **📜 Transaction History**: Har trade ki ek digital raseed (Receipt) banti hai jo Firebase mein save hoti hai.
- **🎨 Modern UI**: Glassmorphism effect aur premium dark/light theme gradients ke sath.
- **⚙️ Reset Account**: Agar user loss mein chala jaye toh wo apna balance aur portfolio wapas reset kar sakta hai.

---

## 🛠️ Tech Stack

- **Frontend**: Flutter (Dart)
- **Backend**: Firebase Auth, Cloud Firestore
- **State Management**: Provider (Clean Architecture)
- **API**: CoinGecko Public API
- **Local Storage/Icons**: Flutter Launcher Icons, Google Fonts
